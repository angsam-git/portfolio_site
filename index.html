MelodyDB

Think of it as an IMDB but for music. This database contains thousands of songs, albums, and artists and information about them. Additional features include allowing users to review and rate songs, as well as administrator accounts to moderate such behavior. PSQL Database filled with data from the Spotify API.

❮ ❯

Selective Repeat
Protocol Emulator

Emulation of the selective repeat protocol. Packets are represented as chars. Program features allow for dropping packets and ACKs on a deterministic and probabilistic basis. Utilizes socket programming and multithreading to send and receive packets.

❮ ❯

InspireWeather

This is an app that provides inspirational quotes along with a full weather report. Features allow for browsing the weather forecast based on both an hourly and daily basis. Each weather card displays with an icon displaying the appropriate weather. Utilizes the DarkSky API and the Google Places API.

Nooks

Apartment hunting website which organizes the apartment hunting process. It allows users to add and track their apartment listings and see them on a map through the Google Geocoding API. It can also serve as a collaborative space which allows users to create groups with their friends. Currently WIP.
